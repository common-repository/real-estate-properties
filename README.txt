== Description ==

A plugin that offers Property custom post type to list properties on your site.

== Installation ==

Instructions for installing the Display Post Metadata Plugin.

1. In your WordPress admin go to Plugins -> Add New.
2. Enter Real Estate Properties in the text box and click Search Plugins.
3. In the list of Plugins click Install Now next to the Real Estate Properties.
4. Once installed click to activate.
